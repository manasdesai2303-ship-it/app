# Goldsmith Management System

## Overview
A full-stack Goldsmith Management System designed to help jewellery businesses manage customers, orders, jewellery inventory, billing, and day-to-day business operations.

## Features
- Customer Management
- Jewellery Inventory Management
- Order Management
- Billing & Invoicing
- Dashboard
- Business Reports

## Tech Stack
- React
- TypeScript
- Node.js
- Express

## Project Structure
```
backend/
frontend/
tests/
README.md
```

## Installation

```bash
git clone <repository-url>
cd app
npm install
npm run dev
```
